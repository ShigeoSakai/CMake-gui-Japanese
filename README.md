# CMake-gui-Japanese
CMake-gui日本語ローカライズ  
(CMake ver3.25から抽出)  

"cmake_ja.qm"ファイルを下記のフォルダにコピーしてください。
（?.??はCMakeのバージョン番号です。)


Windows:  
　C:\Program Files\CMake\share\cmake-?.??\i18n\  
	もしくは、  
　CMakeのインストールフォルダ\share\cmake-?.??\i18n\  

Linux:  
　/usr/share/cmake-?.??/i18n/
	もしくは、  
　/usr/locale/share/cmake-?.??/i18n/
